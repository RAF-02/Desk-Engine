# Desk-Engine
Desk Engine 

Desk Engine is a professional, lightweight desktop overlay utility for Windows. It allows you to place GIFs, images, and videos directly onto your desktop, perfect for dashboard aesthetics, monitoring visual feeds, or just decorating your workspace.

Desk Engine Banner License Platform
✨ Features

    Universal Media Support: Play GIFs, static images (PNG, JPG, WebP), and video files (MP4, WebM, AVI, MOV).
    True Desktop Overlay: Frameless, transparent windows that sit on your desktop.
    Click-Through Mode: Pin an overlay to make it click-through, allowing you to interact with icons or windows behind it.
    Startup Persistence: Pin your favorite windows, and Desk Engine will automatically reload them exactly where you left them when you restart your PC.
    Real-time Window Management: Dynamic window numbering and instant UI updates.
    Global Hotkeys: Close all overlays instantly with a custom global hotkey (Default: Ctrl+/).

🚀 Download & Installation

    Go to the Releases section on the right side of the page.
    Download the latest DeskEngine.exe.
    Double-click to run. No installation required.

    Note: Since this is an unsigned executable, Windows SmartScreen might show a warning. Click "More info" and then "Run anyway" to proceed.

🎮 How to Use
Basic Controls

    Open Media: Double-click the tray icon or use the Control Panel to open files.
    Move: Click and drag the overlay with the Left Mouse Button.
    Menu: Right-click the overlay for context options (Speed, Opacity, Size).
    Pin (📌): Click the Pin button to lock the window.
        Unpinned: Window acts like a normal draggable window.
        Pinned: Window becomes "Click-Through" (transparent to mouse clicks) and stays on your desktop.

Control Panel

The Control Panel allows you to:

    View all active overlays.
    Set default behaviors (Opacity, Speed, Loop).
    Manage Windows Startup settings (Restores pinned windows automatically).

⚙️ Configuration

All settings are saved automatically in:%APPDATA%\DeskEngine\settings.json
